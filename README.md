# OOP Java Workshop - Biratnagar International College

## Overview
This repository contains workshop exercises for the **Introduction to Object-Oriented Programming (4CS021)** course at Biratnagar International College. It covers fundamental and advanced object-oriented programming concepts using Java, progressing from basic syntax to comprehensive OOP principles.

## Course Information
- **Course Code:** 4CS021
- **Course Title:** Introduction to Object-Oriented Programming
- **Institution:** Biratnagar International College
- **Semester:** Second Semester
- **Language:** Java

## Workshop Structure

### Week 1: Java Basics
Introduction to Java fundamentals and basic programming concepts.
- **Files:** Q1.java - Q16.java
- **Topics Covered:**
  - Variables and data types
  - Control flow (if-else, loops)
  - Methods and functions
  - Arrays and basic data structures
  - Input/output operations

### Week 2: Continued Basics
Further practice with fundamental Java concepts.
- **Files:** q1.java - q13.java
- **Topics Covered:**
  - Problem-solving with loops and conditions
  - String manipulation
  - Mathematical operations
  - Nested structures

### Week 3: Introduction to Object-Oriented Programming
Introduction to classes, objects, and basic OOP principles.
- **Key Classes:**
  - `BankAccount` family: Basic class structure, constructors, method overloading, encapsulation
  - `Box.java`: Demonstrating class attributes and methods
  - `Customer.java`: Real-world object modeling
  - `Lamp.java`: Simple class example
  - `Address.java`: Composition and object relationships
  - `UserCreated.java`: Custom user-defined classes
  - `ReverseArray.java`, `SumAverage.java`: Array manipulation
- **Topics Covered:**
  - Class definition and object instantiation
  - Constructors (default and parameterized)
  - Method definition and overloading
  - Access modifiers (public, private)
  - Encapsulation

### Week 4: Inheritance and Polymorphism
Understanding class hierarchies and inheritance.
- **Key Classes:**
  - `Animal.java`, `Animal2.java`: Base class examples
  - `Dog.java`: Inheritance implementation
  - `Parent.java` & `Child.java`: Inheritance hierarchy
  - `Person.java`: Base class for extension
  - `Employee.java`: Real-world object with inheritance
  - `Shape.java`: Polymorphism example
  - `Calculator.java`: Utility class
  - `FinalClass.java`, `FinalMethod.java`: Final keyword usage
  - `SignUp.java`: Application example
- **Topics Covered:**
  - Inheritance (extends keyword)
  - Method overriding
  - Polymorphism
  - The `final` keyword
  - Access modifiers with inheritance
  - Super keyword

### Week 5: Advanced OOP Concepts
Abstract classes, interfaces, and exception handling.
- **Key Classes:**
  - `Abstraction.java`: Abstract class concepts
  - `Interface.java`: Interface implementation
  - `Classes.java`: Advanced class concepts
  - `Exception.java`: Exception handling and custom exceptions
- **Topics Covered:**
  - Abstract classes and methods
  - Interfaces and their usage
  - Exception handling (try-catch-finally)
  - Custom exceptions
  - Loose coupling and high cohesion

### Week 6: Capstone Project
Final integration project combining all learned concepts.
- **Files:** Main.java
- **Description:** A comprehensive project that applies all concepts learned throughout the workshop

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher installed
- A code editor or IDE (VS Code, IntelliJ IDEA, Eclipse, etc.)
- Basic understanding of programming concepts

### How to Run

1. **Navigate to the desired week's directory:**
   ```bash
   cd week_1
   ```

2. **Compile a Java file:**
   ```bash
   javac Q1.java
   ```

3. **Run the compiled class:**
   ```bash
   java Q1
   ```

### File Organization
```
src/
├── week_1/        # Java basics
├── week_2/        # Fundamentals practice
├── week_3/        # Introduction to OOP
├── week_4/        # Inheritance & Polymorphism
├── week_5/        # Advanced OOP concepts
└── week_6/        # Capstone project
```

## Troubleshooting

**Error: "class not found"**
- Ensure you've compiled the file first with `javac`
- Check that you're in the correct directory

**Error: "cannot find symbol"**
- Check for typos in class names and method names
- Ensure all required classes are in the same directory or properly imported

**Error: "public class must be in a file"**
- The public class name must match the filename
